# Minesweeper
 Ein erweitertes Minesweeper-Spiel mit zusätzlichen Fähigkeiten und verbesserten Spielmechaniken. 
 Neben der klassischen Minesweeper-Logik bietet das Spiel eine intelligente 50-50-Erkennung, um schwierige Spielsituationen zu analysieren. 
 Zudem sind spezielle Fähigkeiten integriert, die dem Spieler taktische Vorteile bieten. 
 
  Features & Spielmechaniken: 
 
 -50-50-Erkennung, die riskante Züge hervorhebt 

 -Spezielle Fähigkeiten zur Unterstützung des Spielers

 -Animationen

 -Sounds & Musik
